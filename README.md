# 🧾 Customer Behavior Analysis Dashboard

An end-to-end *Data Analytics Project* analyzing customer purchase behavior using *Python, MySQL, and Power BI*.

---

## 📊 Project Overview
This project explores customer demographics, spending habits, and engagement to uncover actionable business insights.

---

## 🔧 Tools & Technologies
- *Python (Pandas, SQLAlchemy)* – Data cleaning & transformation  
- *MySQL* – Data storage and analysis using SQL queries  
- *Power BI* – Interactive dashboard visualization  

---

## 📁 Project Structure
| File | Description |
|------|--------------|
| customer_behavior_cleaning.ipynb | Python notebook for data preprocessing |
| customer_behavior_queries.sql | SQL queries for key business insights |
| customer_behavior_dashboard.pbix | Power BI dashboard file |
| README.md | Project documentation |

---

## 🧠 Key Insights
- Clothing category generated the *highest revenue*  
- Only *7.3%* customers are subscribed, but they *spend more*  
- *Young Adults* contribute the highest sales among age groups  
- *Top-rated products* and *discount impact* identified using SQL
